# VertexFX Broker Platform

A full-stack trading platform with:
- User frontend (React/Vite)
- Admin dashboard (React/Vite)
- Backend (Node.js/Express + Prisma)

## Features
- User registration & authentication
- Market data & trading interface
- Admin dashboard for managing users & trades
- And more!

## Project Structure
- `user-app/`: User-facing frontend
- `admin-app/`: Admin dashboard frontend
- `vertexfx-backend/`: Node.js backend with Prisma ORM
